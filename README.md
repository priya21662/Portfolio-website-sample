
# HTML Multi-Page Website 

Live Website(Project URL):  
https://priya21662.github.io/Portfolio-website-sample/

This project is from **roadmap.sh** and is designed to help beginners understand how to structure a website using **only HTML**. 

---

##  Project Overview

The goal of this project is to build a simple, multi-page website where each page is connected through a common navigation bar. 


Through this project i learned to 
* Create and link multiple HTML pages
* Use semantic HTML elements correctly
* Add basic SEO meta tags

---

##  Pages Included

The website consists of the following pages:


* **Index (`index.html`)** – Acts as the main navigation hub linking to all pages
* **Homepage (`home.html`)** – Contains personal details 
* **Projects (`projects.html`)** – Displays a list of projects
* **Articles (`articles.html`)** – Contains articles 
* **Contact (`contact.html`)** – Provides contact-related infor
